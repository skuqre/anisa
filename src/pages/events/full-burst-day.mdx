Full Burst Day is a _reoccurring_ event where **rewards are increased** in the following gamemodes:
- Interception
- Simulation Room

Completing/finishing these gamemodes while this event is happening will grant you with more rewards. Due note that this event isn't on a constant schedule and happens on certain occasions. Keep up to date with [NIKKE_en on Twitter](https://twitter.com/nikke_en/) for future schedules! Also, dates for events like these can be found in the Patch Notes in advance.
